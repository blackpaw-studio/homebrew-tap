# Blackpaw-studio Tap

## How do I install these formulae?

`brew install blackpaw-studio/tap/<formula>`

Or `brew tap blackpaw-studio/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "blackpaw-studio/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
